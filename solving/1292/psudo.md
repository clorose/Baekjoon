# 쉽게 푸는 문제

## 문제 링크
- [BOJ 1292](https://www.acmicpc.net/problem/1292)

## 문제 해결 방법

1을 한 번, 2를 두 번, 3을 세 번, 이런 식으로 1 2 2 3 3 3 4 4 4 4 5 .. 이러한 수열을 만들고 어느 일정한 구간을 주면 그 구간의 합을 구하는 것이다.
현재는 1~1000까지 수만 주어지므로 직접 만드는 방법

## 의사코드
```md
A,B = 입력받기
변수 수열 = []
변수 시작 = 1

반복 < B
  수열 += [시작] * 시작
  시작 += 1
반복 끝

출력 수열[A-1:B]
```
