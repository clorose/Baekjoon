# 별 찍기 - 7

## 문제 번호
[백준 2444번](https://www.acmicpc.net/problem/2444)

## 문제 해결 방법
- 별을 찍는 문제이다.
- 별을 찍는 규칙은 다음과 같다.
  - 첫 번째 줄: 1개
  - 두 번째 줄: 3개
  - 세 번째 줄: 5개
  - 네 번째 줄: 7개
  - 다섯 번째 줄: 9개
  - 여섯 번째 줄: 7개
  - 일곱 번째 줄: 5개
  - 여덟 번째 줄: 3개
  - 아홉 번째 줄: 1개

- 여기선 공백이 4, 3, 2, 1, 0, 1, 2, 3, 4개씩 들어간다.

  공백 i번
  별 N-i번
